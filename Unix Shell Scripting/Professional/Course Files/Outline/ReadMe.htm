Course Outline Notes
 

The Course Outline is the PowerPoint slideshow that was used to make up the structure of this course. It is available as a separately viewable document.

In order to view the slideshow, you must be running Windows 95, 98, ME, NT 4, 2000, or XP. In other words, if you're running Windows 3.1 or any non-Windows operating system, you cannot view the stand-alone slideshow separately.

If you currently have Microsoft PowerPoint installed (version 95 or later), then you can go directly to the Slideshow sub-folder and double-click on the Outline.pps.

If you do not currently have PowerPoint installed (or if you have some problem opening the above file), then go to the Slideshow & Viewer sub-folder and double-click on the Outline Setup.exe program.  This will install the free PowerPoint viewer software onto your system (into the folder of your choice) and copy the course outline slideshow to the same folder.  The slideshow should automatically start.
