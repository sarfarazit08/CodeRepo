UNIX Commands
 

The following table lists every UNIX and Bourne shell command mentioned in this course.

Click on a command's name for the GNU man page.  All links are from an online man page source at http://www.rt.com/man

Bold type indicates how a command got its name.

Commands may be marked as follows:

  • Shell - The command is an internal shell command (a shell built-in), meaning it does not exist as separate executable program file
  • Filter - The program is a filter, meaning it is a text-processor - it takes standard input, processes it and produces standard output
  • Admin - The program is typically only available to system administrators (root)
